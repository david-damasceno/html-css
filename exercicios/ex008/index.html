Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em destaque de forma semãntica usando a tag <strong>

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Podemos criar também um texto marcado usando a tag <mark> e mudar a cor com CSS

Texto grade e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Obs, sem semântica

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi inserido depois

Existe também o texto sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subescrito

Para inserir coisas do tipo H2o usando a tag <sub>
